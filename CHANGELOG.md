# Changelog

## v0.2.2

- 运行日志改为实时输出。
- 初始化、提交、清理、检查、pull、push 等关键步骤会边执行边写入日志。
- push / pull 前会显示正在执行的 Git 命令，避免长时间等待时看起来像卡死。
- GitHub 连接失败时会更快定位到具体 Git 命令。
