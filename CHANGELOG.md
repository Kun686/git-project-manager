# Changelog

## v0.2.6

- 修复下载后的 exe 可能无限自我打开的问题。
- 不再直接调用裸命令 `git`，改为先解析真实 Git 安装路径。
- 如果工具自身被命名为 `git.exe`，会阻止启动并提示改名。
- 避免当前目录下错误的 `git.exe` 被当作真正 Git 调用。

## v0.2.5

- 新增 Windows AppUserModelID。

## v0.2.4

- 修复 Windows 打包 exe 后执行 Git 命令时闪出 cmd 窗口的问题。

## v0.2.3

- 新增「GitHub HTTPS 自动转 SSH」开关。
- 修复列表选中状态消失。

## v0.2.2

- 运行日志改为实时输出。

## v0.2.1

- 修复冲突标记检测误报。

## v0.2.0

- 新增远程非空仓库自动拉取合并
- 优化勾选框为苹果风格开关
- 新增提交前误提交扫描
- 新增基础 GitHub Actions CI 生成
- 新增 cache/log 清理
- 新增 .gitignore 管理

## v0.1.0

- 多项目路径管理
- Git 提交、推送、历史查看、回退
